# Generate Affiliate Program Marketing Kit - OpenClaw Skill

Generates a complete affiliate marketing kit with banners, swipe copy, social posts, and tracking guidelines. Use when launching or scaling affiliate programs.

## Installation

```bash
openclaw skill install generate-affiliate-program-marketing-kit
```

> 📖 **Complete setup guide**: https://skillboss.co/skill.md

### Via ClawHub

```bash
clawhub install toby-generate-affiliate-program-marketing-kit
```

### Manual Installation

```bash
git clone https://github.com/TobeyRebecca/generate-affiliate-program-marketing-kit.git
cp -r generate-affiliate-program-marketing-kit ~/.openclaw/skills/generate-affiliate-program-marketing-kit
```
